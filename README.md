# 📊 Análise Exploratória de Dados - COVID-19

Este projeto faz parte do meu portfólio como estudante de **Análise de Dados** e tem como objetivo explorar dados relacionados à pandemia de COVID-19, identificando padrões, tendências e possíveis insights a partir de informações reais.
---

## 🎯 Objetivo do Projeto

Realizar uma **Análise Exploratória de Dados (EDA)** para compreender o comportamento da COVID-19 ao longo do tempo, respondendo questões como:

- Como os casos evoluíram ao longo do tempo?
- Existe padrão de crescimento ou queda?
- Quais períodos foram mais críticos?
- Como se comportam métricas como casos confirmados e mortes?

---

## 🗂️ Dataset

O dataset contém informações relacionadas à pandemia, como:

- Data dos registros  
- Número de casos confirmados  
- Número de mortes  
- Possíveis outras métricas associadas (dependendo da base utilizada)  

---

## 🛠️ Tecnologias Utilizadas

- **Python**  
- **Pandas** → Manipulação e análise de dados  
- **Seaborn** → Visualização estatística  
- **Matplotlib** → Criação de gráficos  
- **Google Colab** → Ambiente de desenvolvimento  

---

## 🔍 Etapas da Análise

### 1. Carregamento dos Dados
- Importação do dataset  
- Visualização inicial dos dados  

### 2. Entendimento dos Dados
- Tipos de dados  
- Identificação de valores nulos  
- Estrutura geral  

### 3. Limpeza e Tratamento
- Ajustes de colunas  
- Conversão de datas  
- Tratamento de valores ausentes  

### 4. Análise Exploratória
- Evolução temporal dos casos  
- Análise de mortes ao longo do tempo  
- Identificação de picos e tendências  

### 5. Visualização
- Gráficos de linha para séries temporais  
- Análises visuais para facilitar interpretação  

---

## 📈 Principais Insights

- Crescimento significativo de casos em determinados períodos  
- Presença de picos que indicam ondas da pandemia  
- Relação entre aumento de casos e aumento de mortes  
- Tendências de estabilização ou queda em certos momentos
